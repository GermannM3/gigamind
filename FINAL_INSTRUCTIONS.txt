## Дальнейшие шаги на Windows

1. **Установка необходимого ПО** (следуйте инструкции в WINDOWS_INSTRUCTIONS.md):
   - Node.js и npm
   - Java 17 Development Kit
   - Android Studio и SDK

2. **Клонирование или копирование проекта**:
   - Либо клонируйте репозиторий и переключитесь на ветку mobile
   - Либо скопируйте файлы вручную

3. **Установка зависимостей**:
   ```
   cd GigaMindMobile
   npm install
   ```

4. **Сборка APK**:
   ```
   cd android
   .\gradlew assembleRelease
   ```

## Если не удается собрать APK

1. **Используйте Expo с облачной сборкой**:
   - Установите Expo CLI: `npm install -g eas-cli`
   - Создайте аккаунт на https://expo.dev
   - Авторизуйтесь: `eas login`
   - Соберите APK: `eas build --platform android`

2. **Используйте Android Studio**:
   - Откройте проект в Android Studio
   - Используйте встроенные инструменты для сборки
